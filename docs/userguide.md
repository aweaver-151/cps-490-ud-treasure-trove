## Creating an Auction

![Failed to Find](/docs/images/BrowsingPage.png)

1. [Log In](#logging-in).
2. Click the Item List button in the top right.
3. Type the title into the title field.
4. Type a description into the second field (optional).
5. Upload an image file to display with the auction (optional).
6. Select an end date and time for the auction.
7. Hit the create button to create your new auction listing.

## Accessing an Auction Page

![Failed to Find](/docs/images/BrowsingPage.png)

1. [Log In](#logging-in).
2. Click the Item List button in the top right.
3. Find an auction in the auction item list with your filters and sorting options.
4. Click on the title of the auction.
